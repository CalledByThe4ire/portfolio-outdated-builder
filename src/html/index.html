Примеры работ
  • Презентация
    которую вы сейчас смотрите

    • Проект сверстан на препроцессорах jade и stylus, в соответствии с методологией БЭМ. Анимация реализована с помощью библиотек animate.css и wow.js. В качестве системы сборки использован gulp.

    • Ссылка на github-репозиторий
  • Pink
    Веб-сайт

    • В качестве препроцессоров использованы jade и stylus, методология именования классов – БЭМ.
      Макет сделан максимально адаптивным и выглядит по-разному для мобильной, планшетной и десктопной версии. Раскладка всех компонентов страницы реализована на flexbox, в качестве сеточной системы выбран lost.
      Для каждого размера и разрешения экрана подготовлен свой комплект изображений; вся декоративная графика собрана в спрайты, которые в свою очередь разделены на векторные и растровые с помощью соответствующих npm-пакетов.
      В качестве системы сборки использован gulp.
      Проект является дипломной работой продвинутого интенсива htmlacademy, по итогам успешной защиты выдан сертификат.
